# HillelFEP7

## Getting Started
I'm using:
```cmd
node --version  // 10.3.0
npm --version   // 6.1.0
```
But tested also on
```cmd
node --version  // 8.11.2
npm --version   // 5.6.0
```

## Installing

Clone from GitHub, run the following commands:

```
git clone git@github.com:skrav4enko/HillelFEP7.git

git checkout hw22

cd hwlayout
```

## Install project dependencies
From hwlayout directory, run:

```
npm install
```

## Running development server
```
npm start
```
## Run productiion build
```
npm run build
```
